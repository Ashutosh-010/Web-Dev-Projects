# Expat Explore Website Clone

A polished HTML/CSS/JavaScript clone of [expatexplore.com](https://expatexplore.com/) with enhanced UI and interactive features.

## Features

### Design
- Professional SVG logo with gradient arrow, gold stripe, and globe
- Plus Jakarta Sans + Montserrat typography
- Scroll progress bar, fade-in reveal animations
- Dark mode toggle
- Glass-style sticky header on scroll
- Tour badges (Best Seller, On Sale, Popular)

### Sections
- Hero with trust badges and destination search
- Animated stats bar (180k+ travellers, 50 countries, etc.)
- Book With Confidence cards
- Filterable tour carousel with wishlist hearts
- As Featured In + blog articles
- Destination chips, Why Travel With Us, FAQ accordion
- Interactive world map, newsletter, full footer

### Interactivity
- Tour quick-view modal
- Wishlist panel (saved in localStorage)
- Live chat widget
- FAQ accordion, tour filters
- Cookie banner, toast notifications
- Back-to-top button
- Currency selector (demo)

## Run locally

```powershell
cd C:\Users\choud\expat-explore-clone
python -m http.server 8080
```

Open **http://localhost:8080** (requires internet for Pexels images).

Or open `index.html` directly in your browser.
